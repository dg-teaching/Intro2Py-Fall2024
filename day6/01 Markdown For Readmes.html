SpikeFinder

Welcome to SpikeFinder, the spikiest tool in your neuroscience toolkit! If you're detecting spikes in calcium imaging files, this package will make your life so much easier.

Features

SpikeFinder has everything you need:

  • Accurate Spike Detection
    - Finds those pesky spikes, even the subtle ones.
    - Built-in thresholding, so you can catch all the spikes.
  • Fast and Efficient
    - Process huge files without breaking a sweat.
    - Optimized for those marathon data analysis sessions.
  • User-Friendly Output
    - Export results in multiple formats: .csv, .json, and even .gif!

Installation

First, ensure you have Python 3.7 or higher. Then, install SpikeFinder via pip:

pip install spikefinder

Or, if you're feeling adventurous, clone the repo and install from source:

git clone https://github.com/fake-neuro-labs/spikefinder.git
cd spikefinder
pip install -e .

Quick Start

  1. Import the package:
import spikefinder
  1. Load your calcium imaging file:
data = spikefinder.load_data("my_calcium_imaging_file.csv")
  1. Run spike detection:
results = spikefinder.detect_spikes(data)
  1. View your spikes:
spikefinder.plot_spikes(results)

...and that’s it!

Configuration Options

Want more control? Customize SpikeFinder to your heart’s content:

  • threshold (float): Set the detection threshold. Defaults to 1.0.
  • window_size (int): Set the smoothing window size in milliseconds. Defaults to 50.

Example usage:

results = spikefinder.detect_spikes(data, threshold=1.5, window_size=100)

Pro Tips

Pro Tip: Lowering the threshold means more spikes detected, but be careful of false positives!

FAQs

Q: Can SpikeFinder handle multiple files at once?
A: Absolutely! Batch processing is baked right in.

Q: What formats does SpikeFinder support?
A: Currently, .csv, .tsv, and .nwb are supported. Let us know if you need more!
